# 求职追踪 Dashboard

求职进度追踪工具，支持投递记录、漏斗图、待办事项和备注。

## 功能

- 投递记录表格：公司、岗位、渠道、状态、优先级、跟进日期
- 求职漏斗图：可视化各阶段转化
- 待办清单：按截止日期排序，支持勾选完成
- 备注区：自由文本记录

## 数据存储

数据保存在浏览器 `localStorage`，key 规范：

| key | 内容 |
|-----|------|
| `qiuzhi_jobs_v1` | 投递记录列表（JSON 数组） |
| `qiuzhi_todos_v1` | 待办事项列表（JSON 数组） |
| `qiuzhi_notes_v1` | 备注文本 |

数据本地保存，刷新不丢失；清除浏览器数据后会重置为示例数据。

## 数据备份与恢复

页面右上角提供两个按钮：

- **导出备份**：将当前所有数据下载为 `qiuzhi_backup_YYYY-MM-DD.json`，文件名带日期，多次导出不覆盖，形成历史版本
- **导入恢复**：选择之前导出的备份文件，确认后覆盖当前数据

建议定期导出备份，防止浏览器清缓存导致数据丢失。

## 使用方式

- 独立打开：直接用浏览器打开 `index.html`
- 本工具为开发内部工具，不在主页展示
