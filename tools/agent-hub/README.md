# Agent 认知全景

AI 产品经理个人求职网站工具模块。

## 功能

通过 4 个 Tab 系统性展示对 Agent 的认知判断深度：

| Tab | 内容 |
|-----|------|
| ⚖️ 框架选型 | 6 个主流框架横向对比 + 条件筛选推荐（可交互） |
| 🏗️ 架构设计 | 4 种多 Agent 拓扑形态（SVG）+ 年报生成序列图 + PM 设计注记 |
| 🏢 企业提效地图 | 6 大企业场景（HR/研发/客服/BI/法务/IT），含痛点→方案→ROI |
| 💡 我的判断框架 | 5 条有立场有数据的 PM 观点，附折叠「反面思考」 |

## 技术

- 纯 HTML + CSS + Vanilla JS，零依赖
- CSS 变量自包含，不依赖 style.css
- 可独立运行：直接打开 `index.html`

## 生成

`index.html` 由 `gen_index.js` 脚本生成（已删除脚本，直接使用生成后的文件即可）。
