# 前沿雷达

追踪 AI 前沿的个人信息源导航 + 精选工具清单。

## 功能描述

精选 AI 产品经理的高质量信息源，中英文分栏展示，体现系统性的信息获取体系。

## 内容板块

- **信息源导航**：中英文分栏，覆盖 Newsletter、技术博客、媒体、社区
  - 中文：36Kr、虎嗅、产品星球、即刻等
  - 英文：a16z、Benedict Evans、Lenny's Newsletter 等
- **精选工具**：个人日常在用的 AI 工具及使用场景简介

## 技术

纯静态 HTML，数据硬编码，无 JS 依赖，可独立运行。

## 访问方式

- 独立打开：`tools/radar/index.html`
- 主页嵌入：工具箱 → 🛰 前沿雷达 Tab（iframe）

## 维护指南

内容全部硬编码在 `index.html` 中，直接编辑对应区块即可：

- **添加信息源**：在 `index.html` 中找到对应分栏的列表，新增 `<li>` 条目
- **添加工具**：在工具列表区块新增卡片
- 更新频率建议：每季度或有重要新增来源时更新
